# 📸 Настройка загрузки фото на Synology NAS

## 📋 Требования
- Synology NAS с DSM 7.0+
- Установленный пакет **Web Station**
- Установленный **PHP** (версия 7.4 или выше)
- Доступ к файловой системе NAS

---

## 🔧 Шаг 1: Включение Web Station

1. Откройте **Package Center** на Synology
2. Найдите и установите **Web Station**
3. При установке выберите PHP (рекомендуется 8.0+)
4. После установки откройте **Web Station**

---

## 🔧 Шаг 2: Создание папки для фото

1. Откройте **File Station**
2. Перейдите в `/volume1/web/`
3. Создайте папку `wedding-photos`
4. ПКМ на папку → **Свойства** → **Разрешения**
5. Добавьте пользователя `http` или `www-data` с правами:
   - ✅ Чтение
   - ✅ Запись
   - ✅ Создание папок

---

## 🔧 Шаг 3: Настройка PHP

1. В **Web Station** перейдите в **Порталы служб**
2. Создайте новый портал или отредактируйте существующий
3. Укажите:
   - **Корневая папка**: `/volume1/web/`
   - **PHP версия**: 8.0+
   - **Профиль PHP**: Создать новый
4. В профиле PHP включите:
   - `file_uploads = On`
   - `upload_max_filesize = 20M`
   - `post_max_size = 25M`
   - `max_file_uploads = 20`

---

## 🔧 Шаг 4: Загрузка файлов сайта

1. Скопируйте все файлы сайта в `/volume1/web/`
2. Структура должна быть:
/volume1/web/ ├── index.html ├── css/ │ └── styles.css ├── js/ │ └── script.js ├── php/ │ └── upload.php └── wedding-photos/ (создана в шаге 2)


---

## 🔧 Шаг 5: Настройка прав доступа

```bash
# Подключитесь к NAS по SSH
ssh admin@your-nas-ip

# Установите правильные права
sudo chown -R http:http /volume1/web/wedding-photos
sudo chmod -R 755 /volume1/web/wedding-photos
🔧 Шаг 6: Проверка работы
Откройте в браузере: http://your-nas-ip/index.html
Перейдите в раздел Фото
Введите имя и загрузите тестовое фото
Проверьте папку /volume1/web/wedding-photos/ — должна появиться папка гостя
🔒 Безопасность
Защита папок с фото
Файл .htaccess автоматически создаётся в каждой папке гостя и запрещает прямой доступ к файлам.

HTTPS (рекомендуется)
В Control Panel → Security → Certificate
Добавьте сертификат Let's Encrypt
Привяжите к вашему домену
Ограничение по IP (опционально)
В upload.php добавьте:

$allowedIPs = ['192.168.1.0/24'];
$clientIP = $_SERVER['REMOTE_ADDR'];
// Проверка IP...
📊 Мониторинг
Все загрузки логируются в файл upload_log.txt в папке wedding-photos.

Для просмотра:

tail -f /volume1/web/wedding-photos/upload_log.txt
❓ Troubleshooting
Проблема	Решение
Ошибка 403	Проверьте права папки wedding-photos
Ошибка 413	Увеличьте post_max_size в PHP
Файлы не загружаются	Проверьте upload_max_filesize
Папка не создаётся	У пользователя http должны быть права на запись
📞 Поддержка
При возникновении проблем проверьте логи:

Web Station: /var/log/httpd-error.log
PHP: /var/log/php-fpm.log
