# 📸 Настройка загрузки фото на Synology NAS

## 📋 Требования
- Synology NAS с DSM 7.0+
- Установленный пакет **Web Station**
- Установленный **PHP** (версия 7.4 или выше)
- Доступ к файловой системе NAS

---

## 🔧 Шаг 1: Включение Web Station

1. Откройте **Package Center** на Synology
2. Найдите и установите **Web Station**
3. При установке выберите PHP (рекомендуется 8.0+)
4. После установки откройте **Web Station**

---

## 🔧 Шаг 2: Создание папки для фото

1. Откройте **File Station**
2. Перейдите в `/volume1/web/`
3. Создайте папку `wedding-photos`
4. ПКМ на папку → **Свойства** → **Разрешения**
5. Добавьте пользователя `http` или `www-data` с правами:
   - ✅ Чтение
   - ✅ Запись
   - ✅ Создание папок

---

## 🔧 Шаг 3: Настройка PHP

1. В **Web Station** перейдите в **Порталы служб**
2. Создайте новый портал или отредактируйте существующий
3. Укажите:
   - **Корневая папка**: `/volume1/web/`
   - **PHP версия**: 8.0+
   - **Профиль PHP**: Создать новый
4. В профиле PHP включите:
   - `file_uploads = On`
   - `upload_max_filesize = 20M`
   - `post_max_size = 25M`
   - `max_file_uploads = 20`
